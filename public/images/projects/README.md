# Project Images

This folder contains images and diagrams for projects.

## Structure

```
images/
└── projects/
    ├── rag-architecture.png
    ├── vitals-dashboard.png
    └── attack-flow.txt
```

## Adding Images

1. Add your project images to this folder
2. Reference them in `src/data/projects.json` using the path `/images/projects/filename.png`

## Image Guidelines

- Use descriptive filenames
- Optimize images for web (use WebP or optimized PNG/JPG)
- Recommended max width: 1200px
- Include alt text in the project artifacts caption
