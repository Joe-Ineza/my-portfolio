# Portfolio Images

## Required Images

### 1. Profile Picture (`profile.jpg`)
- **Location**: `/public/images/profile.jpg`
- **Recommended size**: 600x600px (square)
- **Format**: JPG or JPEG or PNG
- **Description**: Your professional headshot/profile photo
- **Tips**: 
  - Use a well-lit, high-quality photo
  - Keep the background simple
  - The image will be displayed as a circle

### 2. Hero Background (`hero-background.jpg`)
- **Location**: `/public/images/hero-background.jpg`
- **Recommended size**: 1920x1080px or larger
- **Format**: JPG (for better compression)
- **Description**: Background image for the homepage hero section
- **Tips**:
  - Use an abstract, subtle image (tech patterns, gradients, etc.)
  - Avoid busy images that compete with text
  - The image has a white gradient overlay for readability
  - Good options: abstract tech patterns, geometric shapes, code backgrounds

## Adding Your Images

1. Replace the placeholder images with your own
2. Keep the same filenames or update paths in `src/app/page.tsx`
3. Optimize images for web (compress to reduce file size)
